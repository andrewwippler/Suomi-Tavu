
# Suomi Tavu

Based off of the work by [Duukkis](http://www.palomaki.info/apps/haiku/).

## Quickstart

- Clone this Repo
- `cp .env.example .env`
- `php artisan key:generate`
- `php artisan serve`
- POST text to `/tavu` as a form item with the name `textarea`
